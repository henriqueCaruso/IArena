# Tema BRS — Control Room

Tema opcional para competições internas da BR Supply. Foi extraído dos tokens e componentes compartilhados pelo AI OS e pela Mesa BRS.

Use no arquivo de configuração:

```json
{
  "theme": "brs"
}
```

O gerador passa o mesmo starter, tokens, componentes e critérios adicionais para todos os competidores. Assim, a competição mede a abordagem de cada skill dentro da mesma identidade visual.

O tema não inclui logotipo corporativo nem informação interna. Os protótipos devem usar marca textual ou monograma neutro.
