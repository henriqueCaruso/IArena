# Example competition — 3 pricing page hero sections

Three fictional "AI design skills" were asked the same brief: *"Design a hero section for a SaaS
pricing page, product name 'Loop'."* Nobody saw anyone else's output. This is a toy example shipped
with IArena so you can see the review flow before wiring up your own competition.

## Variants

1. **[Variant 1](/gallery/pricing-hero/1)** — minimal, monochrome, huge type
2. **[Variant 2](/gallery/pricing-hero/2)** — playful, gradient, rounded everything
3. **[Variant 3](/gallery/pricing-hero/3)** — dense, feature-comparison table front and center

Try both: select some text inside a variant and click "Comment", **and** toggle "🎯 Element" and click
an icon or empty area. Then click "📋 Copy ALL" to see every comment across all 3 variants aggregate
into one paste-ready block.
